Sushi / No Sushi
A binary image classifier that can detect the presence of sushi in a picture.
The dataset used for training the model is "The Food-101 Data Set" https://www.vision.ee.ethz.ch/datasets_extra/food-101/
The model is created using the fastai library where transfer learning was used with a pretrained resnet50 model. http://fast.ai
The resnet50 model was pretrained with the Imagenet dataset http://www.image-net.org/
